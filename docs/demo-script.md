# SentinelAI Demo Script

## Two-Minute Portfolio Demo

1. Explain that SentinelAI is a lightweight SOC triage assistant that combines detection engineering with AI-style enrichment.
2. Show the dashboard and highlight the number of incidents, severity distribution, and impacted users.
3. Open the highest-severity incident and walk through the evidence timeline, triggered rules, and MITRE ATT&CK mappings.
4. Explain that the backend combines rule-based detections with anomaly scoring using `IsolationForest`.
5. Close by describing the next iteration: hosted LLM summaries, persistent storage, and analyst feedback loops.
